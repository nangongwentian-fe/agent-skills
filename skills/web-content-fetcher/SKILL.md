## 方法一：defuddle.md —— 任意网页转 Markdown

**适用场景**
- 获取任意公开网页的正文内容
- 社交媒体帖子（X/Twitter、微信公众号等）
- 新闻文章、博客、文档页面
- 需要 Markdown 格式输出时

**使用方式**

在目标 URL 前加上 `https://defuddle.md/` 前缀，用 WebFetch 工具抓取：

```
https://defuddle.md/<目标URL>
```

**示例**

```
# 获取 X/Twitter 帖子内容
https://defuddle.md/https://x.com/op7418/status/2033202960843563484?s=46

# 获取新闻文章
https://defuddle.md/https://example.com/some-article

# 获取文档页面
https://defuddle.md/https://docs.anthropic.com/some-page
```

**注意事项**
- 仅支持公开可访问的页面，需要登录的内容无法获取
- 返回内容已去除广告、导航栏等干扰元素，只保留正文
- 图片等媒体资源不会返回，仅返回文字内容
- 部分反爬较强的网站可能无法正常解析
